1. Overview

This is a webpage to solve questions online without the need of a notebook or pen, you can enter the number of questions you want to solve type those questions in textarea provided for questions and answers in textarea for answers. There is also a board provided to draw figures and do rough work, with a stopwatch to see how long it takes you to solve particular number of questions with a save option to save your questions and answers to solve or check later.

2. Ques. No. Page

This is used to take input of how many questions you want to solve, after submitting your input you would be redirected to answersheet page which would have that number of textareas and boards to work. Once you save questions in answersheet you will not be able to change the number of questions here afterwards.

3. Answersheet Page

The answersheet page contains 10 buttons Save & Clear (to save or clear questions from local storage), Start & Stop (for stopwatch), Pen & Eraser (for board), Straight Line, Dotted Line, Free Draw & Draw Circle (to make drawings using respective modes), one input to change colour of pen on board, one for size of pen and one checkbox to draw and erase grid on board. Two textareas & a board for each question to write question and answer in textareas and figures and rough work on board.

4. Answersheet Textarea

The textareas for writing questions and answers are auto resizable according to length of text and produce a typing sound on typing.

5. Answersheet Board

The board has four modes straight line, dotted line, free draw and circle with variable colours and pen sizes. To draw the circle you will get popup to input the circle's x-coordinate, y-coordinate, radius, start angle & end angle on a slider. You can use grid to find iut the x & y coordinates of pixel where you want to put center of the circle. The circle is drawn anti-clockwise with angles 0° at 3'o clock moving clockwise as shown below.

Angles of circle.

For e.g. to draw a circle only in Ist quadrant give start angle = 0° and end angle = 270°.


When using the app on mobile, whenever screen is resized the board is cleared to redraw all the drawings click on pen-size input area and then click outside.
